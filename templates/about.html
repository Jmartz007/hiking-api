About Alaska Hiking API

The Hiking API app was designed by the Clint Campbell and Corey Miller in 2024 as a vehicle for understanding CI/CD and relevant technologies (Docker, Jenkins, automated testing technologies, etc.).

GET /trail

Allows user to get all trail information.

Response
{
"id" : id,
"name" : name,
"city" : city
"description" : description
}

POST /trail

Allows user to submit trail information. The id is set by the database.

Request
Key Value Description
name String The name of the trail being submitted
city String The name of the city where the trail being submitted resides
description String A description of the trail being submitted
